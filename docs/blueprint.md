# **App Name**: Uncertain Cloud

## Core Features:

- Promotional Content Display: 展示关于“内存超售”、“CPU超售”、“城中村数据中心”和“黑互联网”服务的讽刺营销文本。
- Simulated Server Status Dashboard: 显示动态生成的、幽默地糟糕的服务器性能指标（例如，正常运行时间、延迟、数据包丢失），使用 `faker.js` 模拟不可靠的性质。
- Randomized Outage Events: 在服务器状态仪表板上引入零星的、临时的模拟中断或极端的延迟峰值，以模拟文本中提到的“随机延迟”和“断电”。
- Ironic Service Level Agreement Page: 展示一个专门的页面，其中包含与托管服务的条件和限制相关的讽刺条款，概述“服务保证”和“免责声明”。
- Unresponsive Contact Form: 一个简单的联系表单，用户可以填写，暗示或实际延迟响应时间以维持应用程序的核心“不可靠”品牌。
- CDN 502 Simulation: 模拟网站不一定每次都能打开，使用 faker.js 随机生成 CDN 502 错误页面。
- Random Absurd Announcements: 随机生成离谱事件公告。
- Satirical Server Configuration Options: 提供多种可选的服务器配置，但都是讽刺性的配置选项。

## Style Guidelines:

- 视觉方案采用黑暗、有些粗犷的美学。主色调是深沉、柔和的蓝紫色 (#804DCC)，暗示着技术但非常规的核心。背景颜色是相同蓝紫色的极度不饱和、非常深的阴影 (#221F29)，提供了一个清醒而鲜明的画布。鲜艳的洋红色 (#E76BED) 的强调色被谨慎地用于突出显示、警告和号召性用语，增加了一种意想不到的、略微刺耳的触感，强调了该服务的讽刺性质。
- 整个网站用于标题和正文的主要字体是“Space Grotesk”，这是一种无衬线字体，唤起了一种计算机化、技术化和略带前卫的感觉，与应用程序对云服务的批判性和非常规的视角相一致。
- 图标将是极简主义和锐利的，类似于服务器状态、网络连接或警告的基本 UI 元素，重点是在传达故障或不稳定时略微失真或“故障”美学。
- 一个干净但略微非常规的布局，具有用于促销内容和动态服务器状态卡的清晰部分。期望不对称元素和微妙的视觉中断暗示广告的“随机延迟”，而不会妨碍可用性。
- 将在显示关键数据的元素上采用微妙的、非破坏性的故障或闪烁动画，模拟轻微的视觉干扰，以补充不可靠和随机服务行为的主题。
